# Collection of shapes for Libre-office draw.


Changes:
2018-07-28 Created


Can be used for:
-SAP ASE /Replication server diagrams.
-Simple ERD/ table diagrams.


License: inherited from original sources.



**How to add in Libre Draw:**

Open galery (view clip art gallery) , create new theme, import the shape files.



SAP ASE/ Sybase icons(svg files) are taken from:
http://dia-installer.de/shapes/sybase/index.html.
https://gitlab.gnome.org/GNOME/dia/   (GPL v2.0 license)


All others from media wiki (creative commons license).

And of course you can add / combine with own shapes.
